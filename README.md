
##蚂蚁金服奖项项目

###代码规范

每个页面头部必须写上 当前页面名称、功能、作者以及版本信息
```javascript
name: goodsManager
desc: 商品管理页内容
author: 俞雅菲
version：v1.0
```

###提交必看

提交之前必须先 git pull 更新一下代码
再 git add . 提交更改文件
并用git commit -m "此次更新做的操作"
最后在git push提交代码

###文件规范
分别创建自己的文件夹 自己的文件都建在自己的目录下
前台页面 放在view文件夹中的frontSystem中
后台页面 放在view文件夹中的endSystem中
